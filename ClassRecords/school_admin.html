學期資料設定與上傳

請將每項資料貼上/設定完成後，點擊該區塊的「上傳」按鈕獨立更新。必要資料：課程時間 → 學生名冊 → 教師課表。

1. 課程時間表 (每節課起訖時間)

請設定每節課的開始時間和持續分鐘數。

全校教師紀錄管理 (備份/匯入)

說明： 此功能用於備份學校內所有老師的「學生成績 (Gradebooks)」與「日常表現紀錄 (Performance Records)」。
適用情境：期末封存、資料移轉。
注意：資料量較大時，備份與匯入可能需要數分鐘，請勿關閉視窗。

匯出教師紀錄 (備份)

將全校老師的成績與表現紀錄下載為 JSON 檔。

匯入教師紀錄 (合併)

上傳 JSON 檔。此操作為「新增或更新」，
不會刪除現有資料 (Merge Mode)。

學期重設 (危險操作)

功能說明： 此操作用於學期結束時的資料清理。
[會刪除] 全校老師的「成績紀錄」、「日常表現紀錄」與「成績計算規則」。
 [保留不變] 「課程時間表」、「學生名冊」與「全校教師課表」不會被更動。
(若需更新名冊或課表，請使用上方的「上傳」功能直接覆蓋即可)
